
# Optimización de Portafolio de Mercados – SQL Analysis

## 📌 Problema
Evaluar el desempeño de múltiples mercados internacionales para identificar oportunidades de optimización.

## 🎯 Objetivo
Analizar métricas financieras clave para determinar qué mercados generan mayor rentabilidad.

## ⚙️ Qué hice
- Consultas SQL con CTEs y Window Functions
- Cálculo de KPIs: ROAS, LTV/CAC y margen de contribución
- Análisis de desempeño por territorio

## 📊 Resultados
- Identifiqué mercados con mayor retorno de inversión (ROI)
- Detecté oportunidades para optimizar la asignación de presupuesto
- Generé recomendaciones basadas en KPIs financieros

## 🛠️ Tecnologías utilizadas
SQL | CTEs | Window Functions | Análisis de KPIs

## 📁 Estructura del proyecto
- `queries.sql` → consultas principales
- `dataset.csv` → datos utilizados (si aplica)

## 💡 Aprendizajes
Este proyecto fortaleció mi capacidad para analizar datos financieros y construir queries avanzadas para la toma de decisiones.
